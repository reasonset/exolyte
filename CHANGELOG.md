# Beta

## Beginning of the Night 3 (2026-07-02)

-> Tag: `night_is_began`

Instance settings and channel-related features have been added to the admin panel.

A channel operator (chop) feature has been added to channels.
Channel operators can invite users to channels and change channel settings.

If enabled in the instance settings, users can now send invitations to other users and create channels.

A logout feature has been added.

## Beginning of the Night 2 (2026-07-02)

User-driven features have been added.

A direct messaging feature has been added, allowing users to leave or block channels and direct messages.

Users can now search for other users and start direct messages.

A logo has been added.

## Beginning of the Night 1 (2026-07-01)

New features have been added to the Admin Console.

You can now create user invitations from the Admin Console, and users can create accounts by scanning a QR code.

User management features have also been added.

# Alpha

## Twilight (2025-10-28)

-> Tag: `end_of_twilight`

This project was created as a practice exercise in Elixir/Phoenix.

It is a chat application that uses LiveView and relies on CubDB and JSON files for its database.

It features users and channels (the chat rooms to which users belong), and plays a sound when a message is received.

Administrators must use a CLI tool to perform tasks such as creating users and adding them to chat rooms.
